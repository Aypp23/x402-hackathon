# llms.txt
# Free Crypto News API
# Version: 1.0.0
# Updated: 2026-02-02
# Full docs: https://cryptocurrency.cv/llms-full.txt

> Real-time cryptocurrency news aggregator with AI analysis, 200+ sources, and full market data.

## Quick Start

```bash
curl https://cryptocurrency.cv/api/news
```

No API key required. No rate limits. Free forever.

## Core Endpoints

### News
- GET /api/news - Latest crypto news (limit, category, source params)
- GET /api/news/[id] - Single article by ID
- GET /api/search?q={query} - Search news articles
- GET /api/trending - Trending topics and tickers
- GET /api/breaking - Breaking news alerts

### AI Features
- GET /api/ai/sentiment?asset=BTC - Asset sentiment analysis
- GET /api/ai/brief - AI-generated daily market brief
- POST /api/ai/summarize - Summarize article content
- POST /api/ai/debate - Bull vs bear analysis on any topic
- POST /api/ai/counter - Fact-check and counter-argument generation

### Market Data
- GET /api/market/coins - Top cryptocurrencies by market cap
- GET /api/market/coin/{id} - Detailed coin data
- GET /api/market/fear-greed - Fear & Greed Index
- GET /api/market/global - Global market statistics
- GET /api/market/ohlc/{id} - OHLC price data

### Trading
- GET /api/signals - AI trading signals
- GET /api/whale-alerts - Large transaction alerts
- GET /api/liquidations - Exchange liquidation data
- GET /api/funding - Perpetual funding rates
- GET /api/arbitrage - Cross-exchange arbitrage opportunities

### Real-time
- GET /api/sse/news - Server-sent events for live news
- GET /api/sse/prices - Live price updates
- WebSocket: wss://cryptocurrency.cv/ws

## Response Format

All endpoints return JSON:

```json
{
  "data": [...],
  "meta": {
    "total": 100,
    "page": 1,
    "limit": 20
  }
}
```

### Python
